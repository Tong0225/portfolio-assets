# 作品集素材存储

## 文件夹结构

- `images/` - 图片文件（作品封面、截图等）
- `pdfs/` - PDF文档（简历、作品文档等）

## 使用方式

上传文件后，获取链接格式：

### 图片
```
https://cdn.jsdelivr.net/gh/Tong0225/portfolio-assets@main/images/文件名.jpg
```

### PDF
```
https://cdn.jsdelivr.net/gh/Tong0225/portfolio-assets@main/pdfs/文件名.pdf
```

## 注意事项

- 单文件不超过 100MB
- 支持 jpg/png/gif/pdf 等格式
